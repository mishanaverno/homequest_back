# HomeQuest Back end
**API:**
| Verb | Path                 | Action   | Description |
|------|----------------------|----------|-------------|
| GET  | /                    | Closure  |            |
| POST | /gang                | store    |            |
| GET  | /gang/{id}           | show     |            |
| GET  | /gang/{id}/heroes    | heroes   |            |
| PUT  | /gang/{id}           | update   |            |
| POST | /hero                | store    |            |
| PUT  | /hero/{id}           | update   |            |
| GET  | /hero/{id}           | show     |            |
| POST | /quest               | store    |            |
| PUT  | /quest/{id}          | update   |            |
| PUT  | /quest/{id}/progress | progress |            |
| GET  | /quest/{id}          | show     |            |
